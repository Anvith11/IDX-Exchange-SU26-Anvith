# Weekly Deliverables

This folder contains all weekly deliverables for the IDX Exchange project.

## Structure

- **Week [Number]** - Each week's folder contains deliverables, progress reports, and documentation

## How to Use

1. Create a new folder for each week (e.g., `Week 1`, `Week 2`, etc.)
2. Organize deliverables by category within each week folder
3. Include a summary document at the beginning of each week

## Categories (Optional)

- `Documentation` - Technical documentation, guides
- `Code` - Source code, implementations
- `Reports` - Weekly status reports, progress updates
- `Artifacts` - Diagrams, designs, assets
